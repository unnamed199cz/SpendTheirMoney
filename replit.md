# Static Website Project

## Overview
A simple static HTML/CSS website with a fun "spend money" theme featuring plushie items.

## Project Structure
- `index.html` - Main HTML page
- `style.css` - Stylesheet
- `README.md` - Project documentation

## Development
The site is served using Python's built-in HTTP server on port 5000.

## Deployment
Configured for static deployment, serving files from the root directory.
